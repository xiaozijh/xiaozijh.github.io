Publicado el 2019-08-03 Comentarios: 0 Visitas:

坚持，热爱，生活。

电压 - 电路中2点电势差。
电流 - 电路中导体2段不为0，则有。
功率 - p = vi （w）。
欧姆定律 - u = IR。
串联电流，电压之和为元件间总和，电流相等，首尾相连；并联电流为分路电流总和，电压相等，头头相连。
串联电阻总和 - 每个阻值相加；并联电阻总和 - 每个电阻阻值倒数相加。
基尔霍夫电流定律 - 流入节点电流等于流出该节点电流相等。
原电池 - 1.5v倍数。 (其他能转为电能)
蓄电池 - 1.2v倍数。 (ah)
额定电压 = 电路使用电压。
适配器输出最大电流 > 电路使用电流。
单极性电源 - VCC，GND。
双极性电源 - VCC，GND，-VCC
焦耳定律 - Q = I^2RT。(J)
电阻 - 阻值大小阻止电流大小。
电阻额定功率 - P = I^2R。
热敏电阻 - PTC(温度升高，阻值升高)，NTC（温度升高，阻值降低）非线性。
光敏电阻 - 光线强阻值大，光线弱阻值弱。
电阻 - 串联分压；并联分流。
频率 - F = 1 / T。(周期性变化的频率，每秒出现相同完整的波形的次数,单位hz)
电流 - 移动的是电子。
电容 - 充放相当于短路 - 电子流动；电容电压不能突变。
***隔直通交信号
*存储电荷
*F - mf - uf - nf - pf （千进制）
*电路超过额定电压则损毁
*漏电流 - uA
*无极性 - 云母（高频），瓷片，涤纶
*极性 - 电解，钽电容
*串联 - 总和为每个元件倒数相加的
*并联 - 总和为每个元件相加
*电荷 - Q = V C （V = dV/dT - 每秒变化电压）。
*电荷存储电场能量 - W = (Q . V) / 2
*容抗 - Xc - 1 / (2πfc) hz , f ,Ω。
*电容两端 - 相移是最大值为电流领先电压90度，变化不同。
*电阻 - 相移相同，变化相同。
*RC电路 - 实现不同的频率。
*充电 - 电容两端电压时间曲线，特殊时刻 t = rc。
*耦合 - 信号的连接和传递
